A 100% free RAG-powered PDF chatbot that lets you upload documents and ask questions using natural language. Built with Streamlit for the UI, LangChain for document processing, BM25 for retrieval, and Groq (LLaMA 3.1) as the LLM backend.

Features:
 Multi-PDF upload and indexing
 Streaming chat responses
 BM25 keyword retrieval with chunk previews
 Source citations with page numbers
 Chat history export
 Live activity log and token usage
 Dark mode UI with sidebar file manager

 Stack: Streamlit · LangChain · Groq · LLaMA 3.1 · BM25 · PyPDF
 Cost: Completely free to build and deploy
